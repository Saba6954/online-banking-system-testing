# online-banking-system-testing
Manual testing project for an Online Banking System including Test Plan, Test Cases, Bug Report, RTM and Test Summary Report.

Project Overview

This repository contains the manual testing artifacts for an Online Banking System. The objective of this project is to verify that all banking functionalities work as expected, identify defects, and ensure a secure and user-friendly experience.
The testing was performed using manual testing techniques following the Software Testing Life Cycle (STLC).
